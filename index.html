Neue Montreal Font CDN

A modern sans-serif typeface hosted on GitHub Pages

By Boluwatife Omotoyinbo

View Demo GitHub Repo

Quick Start

Add this line to your HTML head:

<link rel="stylesheet" href="https://blacip.github.io/neue-montreal/css/font.min.css">

CSS Usage

.my-text { font-family: 'Neue Montreal', sans-serif; font-weight: 400; /* 300, 400, 500, 700 */ }
